## 🎯 프로젝트명: Campus Parking Guide

**"학교 방문객과 초보 운전자를 위한 드론 매핑 기반 AI 주차 가이드"**

## 타겟 유저

- 학교 지리에 익숙하지 않은 방문객(행사, 기숙사 입사)
- 주차 공간을 못 찾아 헤매는 학생/교직원

## 기획 의도

- 기존 지도 앱(네이버/카카오)은 학교 내부의 **세밀한 진입로**나 **임시 주차 공간** 정보가 부족함
- 드론의 **시각적 정보(Visual)** 와 Unity의 **3D 정보**, AI의 **분석 정보**를 결합해 직관적인 정보를 제공

## 🔧 공통 기능: 데이터 분석 및 시각화

### ✅ Streamlit 기반 데이터 분석 대시보드

- **목적:** 드론 데이터 및 주차장 데이터를 수집한 후, 분석 결과를 직관적으로 시각화하여 팀원들과 공유
- **구현 내용:**
    - 드론 이미지 데이터 및 주차장 좌표 데이터 로드
    - 학교 주차장 입출차 현황 데이터를 바탕으로 인사이트 도출
    - 시간대별 입출차 계산 및 시각화 (차트, 그래프)
    - 인터랙티브 맵을 통한 주차장 위치별 데이터 탐색
- **기술 스택:** Python, Streamlit, Pandas, Matplotlib/Plotly
- **산출물:** 웹 기반 대시보드로 팀원 누구나 접근 가능한 분석 환경 구축
